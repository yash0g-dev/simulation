# Physics Shorts Simulations

Creative physics simulations built with Python + Pygame.

This project contains:

- Multi-ball simulations
- Obstacle race simulations
- Stair music simulations
- Collision sound systems
- Motion trails and visual effects

---

# Requirements

Install:

- Python 3.11+
- Git
- FFmpeg (optional, for video exporting)

---

# Clone Repository

```bash
git clone https://github.com/yash0g-dev/simulation.git
cd shorts

#Linux
python3 -m venv venv
source venv/bin/activate

#Windows
python -m venv venv
venv\Scripts\activate

#install dependency 
pip install -r requirements.txt

#run 
python3 main.py
```
